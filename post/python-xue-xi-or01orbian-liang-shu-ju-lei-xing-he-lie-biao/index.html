Python学习｜01｜变量、数据类型、注释

· 2020-07-19 · # Python

变量、数据类型和列表。

变量

命名规则

  • 变量名只能包含字母、数字和下划线。变量名可以是字母或则下划线开头，不能是数字开头。
  • 变量名不能包含空格，但是可以使用下划线分隔单词。
  • 不能使用 Python 关键字和函数名作为变量名，即 Python 保留用于特殊用于用途的单词。
  • 变量名应该简短又具有描述性。
  • 慎用小写字母l和大些字母O，因为可能被人看错成数字1和0。

数据类型

字符串

字符串就是一系列字符。在python中，用引号括起来的就是字符串，其中引号可以是单引号或则双引号，如

    "This is a string."
    'This is anther string.'

灵活使用单引号和双引号能够在字符串中包含引号和撇号。

大小写转换

方法 用途 示例
title() 将首字母大写的方式展示每个单词，即将每个单词首字母都改为大写。
upper() 字符串转为大写
lower() 字符串转为小写

存数据时，lower方法很有用，很多时候不能依赖用户提供正确的大小写，可以先转为小写存储，再首字母大写显示。

合并（拼接）

python使用加号（+）来完成字符串拼接。

first_name = "ada"
last_name = 'lovelace'
full_name = first_name + " " + last_name
print(full_name)

使用制表符（\t）或则换行符（\n）添加空白

空白泛指任何非打印字符，如空格、制表符和换行符。

删除空白

strip ：剔除两端的空白
rstrip : 剔除末尾的空白
lstrip : 剔除开头的空白

示例：

message = " hello kitty "
message.strip()
message.lstrip()
message.rstrip()

以上代码的输出是：

'hello kitty'
'hello kitty '
' hello kitty'

数字

整数

支持加（+）减（-）乘（*）除（/）运算。
乘方运算（**）.
此外，python支持运算次序，可以通过括号修改运算次序。

浮点数

python中凡是带有小数点的数字都是浮点数。
小数点可以出现在任何位置。
注意：小数点后小数位是不确定的。

>>> 0.2 + 0.1
0.30000000000000004

函数str()避免类型错误

将非字符串类型值转为字符串。

year = 2020
message = "This is " + str(year) + "AD."
print(message)

输出结果： This is 2020AD.

注释

在Python中，注视用井号（#）标示。井号后面是注视内容，会被python解释器忽略。
在代码中要编写清晰、简洁的注视。
